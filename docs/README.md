# PartyPad documentation

Long-lived design notes and implementation roadmaps live here so project
direction is not dependent on issue trackers or prior agent conversations.

## Roadmaps

- [`roadmaps/umd-self-service.md`](roadmaps/umd-self-service.md): turn the
  current owner-operated online prototype into a self-service, cross-platform
  release that any UMD Dolphin user can install without contacting the service
  operator.

Roadmaps describe intended work, not current guarantees. The root
[`README.md`](../README.md) remains the source of truth for behavior users can
run today, and [`AGENTS.md`](../AGENTS.md) contains mandatory agent constraints.
